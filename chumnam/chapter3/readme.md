# 연산자우선순위

1. 다항연산자 : !(논리부정), ~(비트부정), + -(부호), ++ -- (증감연산자)
2. 산순연산자 : *(곱하기), /(나누기), %(나머지연산자), +(더하기), -(빼기)
3. 시프트연산자 : <<, >>, >>>
4. 관계연산자 :  <, <=, >, >=, ==, !=
5. 논리연산자 : 비트논리연산 &, ^, | 일반논리연산 &&, ||
6. 조건연산자 : ?, :
7. 대입연산자 : =, +=, -=, *=, /=, %=, <<=, >>=, >>>=, &=, ^=, |=

다항연산자에서 -> 대입연산자순으로 순위가 낮아짐
